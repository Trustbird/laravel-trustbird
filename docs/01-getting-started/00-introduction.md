# Trustbird Laravel Core

Trustbird is an open-source trust operating system for organisations that want to build and maintain an Information Security Management System (ISMS).

The Laravel Core package contains the foundational building blocks used throughout Trustbird, including domain models, actions, events, commands and configuration.

## Current status

Trustbird is currently in active alpha development.

APIs, database structures and package boundaries may change before the first stable release.

## Core principles

Trustbird is built around a few key principles:

* Business language before compliance language.
* Open and self-hostable by default.
* Auditability by design.
* Frameworks are outputs, not the primary user experience.
* AI assists, humans decide.

## Available domains

### Workspaces

The primary building block for multi-tenancy. Workspaces allow you to group people, assets, and other resources into distinct organizational units.

### People

Track employees, contractors and other individuals that are part of an organisation's trust and compliance scope.

### Teams

Group people within your organization into departments, squads or projects.

### Assets

Manage physical and virtual assets, devices, systems and data stores.
