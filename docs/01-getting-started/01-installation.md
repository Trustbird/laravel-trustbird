# Installation

## Requirements

* Laravel 12 with PHP 8.2 or higher
* Laravel 13 with PHP 8.3 or higher

## Install the package

```bash
composer require trustbird/laravel-trustbird:^0.1@alpha
```

## Run migrations

```bash
php artisan migrate
```

## Verify installation

```bash
php artisan about
```

You should see Trustbird listed among the registered packages.

## Alpha notice

Trustbird is currently in active alpha development.

Breaking changes may occur between releases until version 1.0.
