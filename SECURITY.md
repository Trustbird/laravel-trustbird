# Security Policy

## Supported Versions

Trustbird is currently in active alpha development.

Security fixes will generally be applied to the latest supported release.

| Version | Supported |
|----------|----------|
| 0.x | Yes |
| < 0.x | No |

## Reporting a Vulnerability

Please do not disclose security vulnerabilities through public GitHub issues.

Instead, report vulnerabilities by email:

security@trustbird.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested remediation (if known)

We will acknowledge receipt of your report as soon as possible and aim to provide regular updates during the investigation.

## Responsible Disclosure

We kindly request that security researchers:

- Allow reasonable time for remediation before public disclosure.
- Avoid accessing, modifying, or deleting customer data.
- Avoid actions that could impact service availability.
- Act in good faith.

We appreciate responsible security research and will provide attribution where appropriate.
